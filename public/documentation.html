MyFlix

The web application provides users with access to information about different movies, directors, and genres. Users are able to sign up, update their personal information, and create a list of their favorite movies.

REQUEST URL HTTP/METHOD
Get a list of all the movies /movies GET None A JSON object holding data about all the movies
Return data about a single movie by name /movies/[title] GET None A JSON object holding data about a single movie, containing an name, director, description and an image.
Get data about the genre by name /movies/[title]/genre GET None A JSON object holding the genere of a single movie.
Get data about a director by name /movies/[title]/director GET None A JSON object holding data about the director of a single movie.
Add new users /users POST A JSON object holding data about the user to add. A JSON object holding data about the user that was added. e.g. { "Username": "ryan", "Password": "47474", "Email": "ryan@gmail.com", "Birthday" : "1992-04-19" }
Update user info /users/[user] PUT A JSON object holding data about the information to update. A JSON object holding data about the information updated. e.g. { "Username": "ryan", "Password": "47474", "Email": "ryan@gmail.com", "Birthday" : "1992-04-19" }
Allow users to add a movie from their list of favorites. /users/[user]/Movies/[MovieID] POST None. A JSON object holding data about the user's favorite movie that was added.
Remove a movie from their list of favorites. /users/[user]/Movies/[title] DELETE None A text message indicating whether the movie was successfully removed from the favorites.
Remove user users/[user] DELETE None A text message indicating whether the user was successfully removed.
